## Running local environment

use git bash to run this.

install the dependencies:

```bash
mix deps.get
```

copy the .env.example to .env file with:

```bash
cp .env.example .env
```

and put the token. After that, just run ./runlocal.sh like this:

```bash
./runLocal.sh
```
